# bread

A WIP shoddy platform layer that you can bind your own renderer to, made for educational purposes, written in C using [htils](https://github.com/h4rldev/htils).

## Status

Unusable/under development, X11 and Wayland is a lotta boilerplate, so you gotta wait a little.
